sudo systemctl start docker
sudo systemctl status docker


전체 컨테이너 시작
docker-compose up -d

중지
docker-compose down

특정서비스만 재시작
docker-compose restart mqtt-broker

실행중 컨테이너
docker ps

특정 서비스 로그
docker-compose logs -f mqtt-broker

컨테이너 내부 bash 접근
docker exec -it my-mqtt-broker-auth /bin/sh

docker 볼륨 삭제
docker volume rm postgres_data

docker 볼륨 전체삭제
docker volume prune

로그확인
sudo docker logs -f hiteen-postgresql-1
